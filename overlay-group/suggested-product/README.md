# Suggested Product

<figure><img src="../../.gitbook/assets/Screenshot_119.png" alt=""><figcaption></figcaption></figure>

|              |                                                                           |
| ------------ | ------------------------------------------------------------------------- |
| Enable       | Enable to show the suggested product section.                             |
| Color scheme | Select any color scheme defined in the theme settings > Colors > Schemes. |
| Text         | Add the text for the section.                                             |
| Position     | Select the position to display the section. (Left, Right)                 |
