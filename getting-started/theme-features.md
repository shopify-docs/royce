# Theme Features

**Shopify OS 3.0 Framework**

* Built on Shopify Online Store 3.0 for faster performance and better flexibility
* Modular sections and blocks are available across all pages

**Layout & Customization**

* Multiple page layouts to create varied store structures
* Flexible sections with block combinations for fully customized layouts
* Multiple header styles to suit different branding needs
* Drag-and-drop blocks to easily rearrange homepage content
* Custom content blocks to showcase images and unique content
* Highly customizable theme settings for colors, fonts, and backgrounds — no coding required

**Responsive & User Experience**

* Fully responsive design optimized for desktop, tablet, and mobile devices
* Smooth browsing experience across all screen sizes

**Advanced Store Features**

* AJAX product filters for colors and other attributes
* AJAX cart for adding products without leaving the page
* AJAX wishlist to save favorite items
* Built-in deal timer for promotions and limited-time offers
* Product sorting options on collection pages (price, alphabetical, newest, etc.)

**Global & Marketing Ready**

* Built-in multi-currency support (AUD, CAD, USD, GBP, JPY, EUR, NZD, INR)
* Newsletter subscription forms (pop-up and standard)
* Social sharing support for X (Twitter), Facebook, Instagram, and Pinterest

**SEO Optimized**

* Uses Shopify’s latest SEO standards
* Structured product data optimized for Google rich snippets and search visibility
