# Header block

A Header Block is used to display the main header content of the store. It helps organize branding and key actions at the top of the page.

<figure><img src="../../../.gitbook/assets/Screenshot_16 (1).png" alt=""><figcaption></figcaption></figure>

|                   |                                                                              |
| ----------------- | ---------------------------------------------------------------------------- |
| **Layout**        |                                                                              |
| Desktop width     | <p>Set the content width for desktop view.</p><p>(Fit, Fill, or Custom)</p>  |
| Desktop alignment | <p>Align text content for desktop view.</p><p>(Left, Center, Right)</p>      |
| Mobile alignment  | <p>Align text content for mobile view.</p><p>(Left, Center, Right)</p>       |
| Padding           | Adjust the vertical padding of the section to control spacing. (Top, Bottom) |
