# Content Block

<figure><img src="../../../.gitbook/assets/Screenshot_5.png" alt=""><figcaption></figcaption></figure>

|                |                                                                                                                                |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Image          | Add the image for the content block.                                                                                           |
| Icon           | Select the icon for dropdown.                                                                                                  |
| Divider        | Enable to show the divider between contents block.                                                                             |
| Typography     | Select the option from the typography. According to the selected option you will be adjusted to the font family and font size. |
| Padding Top    | Adjust top spacing for the section.                                                                                            |
| Padding Bottom | Adjust bottom spacing for the section.                                                                                         |
