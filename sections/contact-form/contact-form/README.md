# Contact form

The Contact Form allows customers to send inquiries or messages directly from your store. It helps collect essential details such as name, email, and message, making communication easy and organized.

<figure><img src="../../../.gitbook/assets/Screenshot_14 (1).png" alt=""><figcaption></figcaption></figure>

|                   |                                                                                                                                                                             |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Form**          |                                                                                                                                                                             |
| Style             | Select the form style. (Form, Image, Map)                                                                                                                                   |
| Input style       | Select the input style of the form.                                                                                                                                         |
| **Button**        |                                                                                                                                                                             |
| Style             | <p>Select the button style. (Primary, Secondary, Link )<br>Note: According to the selected button style, the button background and hover background color get changed.</p> |
| Desktop width     | Set the button width for desktop view. (Fit, Fill)                                                                                                                          |
| Mobile width      | Set the button width for mobile view. (Fit, Fill)                                                                                                                           |
| Desktop alignment | Align text content for desktop view. (Left, Center, Right)                                                                                                                  |
| Mobile alignment  | Align text content for mobile view. (Left, Center, Right)                                                                                                                   |
| Padding           | Adjust the padding range around the section. (Top, Bottom, Left, Right)                                                                                                     |
