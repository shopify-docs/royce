# Collection card

A Collection Card is used to showcase a product collection. It helps customers navigate and explore categories easily.

<figure><img src="../../../.gitbook/assets/Screenshot_13 (1).png" alt=""><figcaption></figcaption></figure>

|                      |                                                                           |
| -------------------- | ------------------------------------------------------------------------- |
| **Text**             |                                                                           |
| Placement            | Select the content placement. (On image, Below the image)                 |
| Alignment            | <p>Choose the alignment of the button.<br>(Left, Center, Right)</p>      |
| **Appearance**       |                                                                           |
| Inherit color scheme | Inherit the color scheme from the global theme settings.                  |
| Color scheme         | Select any color scheme defined in the theme settings > Colors > Schemes. |
| Borders              | Select the border style.                                                  |
| Thickness            | Adjust the border thickness.                                              |
| Opacity              | Adjust the opacity range.                                                 |
| Corner radius        | Adjust the corner radius of the border.                                   |
