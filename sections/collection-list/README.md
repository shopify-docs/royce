# Collection List

The **Collection List** section allows you to display a list of collections in a visually appealing way, making it easy for customers to explore various categories or product groups.

1. **Go to** Shopify Admin > Online Store > Themes.
2. **Click** Customize on your live theme.
3. In the theme editor, **click** Add Section > **Collection list**.

<figure><img src="../../.gitbook/assets/Screenshot_12 (1).png" alt=""><figcaption></figcaption></figure>

|                     |                                                                              |
| ------------------- | ---------------------------------------------------------------------------- |
| Full width          | Enable for full-width view. (Controls the container width)                   |
| Color scheme        | Select any color scheme defined in the theme settings > Colors > Schemes.    |
| Collections         | Select collection.                                                           |
| **Layout**          |                                                                              |
| Collection per page | Adjust the range to display the total count of the products.                 |
| Horizontal gap      | Adjust the spacing range between the collection card.                        |
| Vertical gap        | Adjust the spacing range top and bottom of the collection card.              |
| **Column settings** |                                                                              |
| Desktop             | Control the number of products shown per row on desktop view.                |
| Mobile              | Control the number of products shown per row on mobile view.                 |
| Padding             | Adjust the vertical padding of the section to control spacing. (Top, Bottom) |
