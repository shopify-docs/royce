# Collection Showcase Block

Displays products from a selected collection, used to showcase featured or curated items within a section.

<figure><img src="../../.gitbook/assets/Screenshot_3.png" alt=""><figcaption></figcaption></figure>

|            |                                                                                                                                |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Collection | Select the collection from the created collection.                                                                             |
| Image      | Add the image for the collection showcase block.                                                                               |
| Typography | Select the option from the typography. According to the selected option you will be adjusted to the font family and font size. |
| Link       | Add the URL for the collection showcase block.                                                                                 |
