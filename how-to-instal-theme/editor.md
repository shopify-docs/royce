# Theme Installation

Before installing the theme, please unzip "**Classique-shopify-theme.zip**" and upload the "**Classique.zip**" file to Shopify to avoid the **"layout/theme.liquid" missing template error.**

### **Theme** Extractions <a href="#preparing-the-theme-package" id="preparing-the-theme-package"></a>

1. Download the package from [Themeforest.net](https://themeforest.net/) (it will be in ZIP format).
2. Find the file **"themeforest-xxxxxx-royce-shopify-theme.zip"** in your folder.
3. Extract the contents of the **ZIP** package you just downloaded.
4. Upload the **royce.zip** file to your Shopify store.

<figure><img src="../.gitbook/assets/Screenshot_7 (1).png" alt=""><figcaption></figcaption></figure>

### Upload Theme Through Shopify Admin Panel

<figure><img src="../.gitbook/assets/Screenshot_72 (1).png" alt=""><figcaption></figcaption></figure>

1. **Log in** to the Shopify Panel.
2. Click **"Online Store"** from the left-hand menu.
3. Click **"Add theme" > "Upload zip file."**
4. Select and upload the **royce.zip** file. The theme will then be uploaded to your store.
5. Once the upload is complete, click **"Actions"** > **"Publish."**
