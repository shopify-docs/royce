# Badge

In the badges section, you can customize the **Badge position** and **Corner radius,** and also you can change the **Background Colour** theme and **Text color** for the **Sale, Sold, and a New badge.**

<figure><img src="../.gitbook/assets/Screenshot_11 (1).png" alt=""><figcaption></figcaption></figure>

| Text case                | Use the Text case setting to adjust the text style. ( Uppercase, Default )                                                                                        |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Corner radius            | Adjust the corner radius for the badge field.                                                                                                                     |
| Sale, New, Soldout badge | <ul><li>Show badge: Enable badge option.</li><li>Background: Pick the color for the badge background.</li><li>Color: Pick the color for the badge text.</li></ul> |
