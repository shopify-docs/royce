# Typography

With the Typography settings, you have the freedom to personalize the font styles, sizes, and letter casing for your entire store.

1. **Go to** Shopify Admin > **Online Store > Themes**.
2. Click **Customize** on your active theme.
3. In the Theme Editor, click **Theme Settings > Typography** .

**Headings**

* Configure the **font type** for all headings used in the theme.
* Choose from available font styles to match your brand identity.
* You can also use the custom font.
* To change the default font size for headings, use the Font size scale to select the default font size. The minimum font size is 100%, and the maximum font size is 125%.

**Body**

* The theme primarily uses the **body font** as the main font type.
* Customize the **font type** to ensure readability and consistency across your store.
* You can also use the custom font.
* To change the default font size for the body, use the Font size scale to select the default font size. The minimum font size is 100%, and the maximum font size is 125%.
