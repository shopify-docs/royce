# Input Field

In Shopify, **Input Settings** refer to customizing forms and input fields such as contact forms, search bars, login forms, and checkout fields.

* Go to **Online Store**> Click **Theme Settings** icon.
* Locate **Input Fields** on the left-side.
* Click **Save.**

<figure><img src="../.gitbook/assets/Screenshot_8 (1).png" alt=""><figcaption></figcaption></figure>

| Thickness     | Adjust the thickness of the input field border. |
| ------------- | ----------------------------------------------- |
| Corner radius | Adjust the corner radius of the input field.    |
