# Price

Displays the product price, including sale and compare-at pricing when applicable.

<figure><img src="../../../.gitbook/assets/Screenshot_49.png" alt=""><figcaption></figcaption></figure>

|                |                                                                                                                                |
| -------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| Compare price  | Enable to show Actual price.                                                                                                   |
| **Appearance** |                                                                                                                                |
| Typography     | Select the option from the typography. According to the selected option you will be adjusted to the font family and font size. |
| Padding        | Adjust the vertical padding of the section to control spacing. (Top, Bottom)                                                   |
